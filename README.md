<p>
<b>InsightIQ</b> is an interactive <b>EDA (Exploratory Data Analysis) automation tool</b>

<b>data summaries, missing value analysis, visualizations, and AI-generated insights</b>.

by automating repetitive EDA tasks through a clean web interface.
